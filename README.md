#  React Weather App

A modern and responsive weather application built with **React** and **Redux Toolkit**.  
Users can search for any city and get real-time weather information using a weather API.

---

##  Features

-  Search weather by city name  
-  Real-time temperature & weather data  
-  Built with React (Vite)  
-  State management using Redux Toolkit  
-  Fully responsive (Mobile, Tablet & Desktop)  
-  Clean and modern UI  

---

##  Tech Stack

- React JS
- Redux Toolkit
- JavaScript (ES6+)
- CSS3
- Weather API
- Vite
